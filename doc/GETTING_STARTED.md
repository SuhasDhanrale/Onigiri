# Getting Started

This guide will help you set up the development environment for Shogun Tower Defense.

## 1. Prerequisites
- **Node.js**: Version 18 or higher.
- **npm**: Version 9 or higher.

## 2. Installation
Clone the repository and install the dependencies:
```bash
npm install
```

## 3. Development Server
To start the local development server with hot-reload:
```bash
npm run dev
```
By default, the game will be available at `http://localhost:3001` (or the next available port).

## 4. Building for Production
To create a production-ready bundle in the `dist/` folder:
```bash
npm run build
```

## 5. Preview Production Build
To preview the build locally:
```bash
npm run preview
```
