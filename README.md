# Shogun Tower Defense

A modular, high-performance Tower Defense game built with React and HTML5 Canvas, featuring a traditional Ukiyo-e aesthetic.

## 🚀 Quick Start
To get the game running locally, see our [Getting Started Guide](./doc/GETTING_STARTED.md).

## 📖 Developer Documentation
We have organized the project's documentation in the `doc/` folder:
- **[Developer Guide](./doc/DEVELOPER_GUIDE.md)**: Technical overview of the architecture, state management, and systems.
- **[System Map](./doc/SYSTEM_MAP.md)**: Detailed breakdown of the project's file structure and module responsibilities.
- **[Getting Started](./doc/GETTING_STARTED.md)**: Installation and development setup.

## 🛠️ Built With
- **Vite**: Frontend tooling and HMR.
- **React**: UI components and lifecycle management.
- **HTML5 Canvas**: High-performance game loop and custom rendering.
- **mitt**: Light-weight event bus for cross-system communication.
