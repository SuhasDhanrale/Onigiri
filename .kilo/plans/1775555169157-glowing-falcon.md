### Step 4: Wire SumiResultScreen into combat victory (IN PROGRESS)
**File:** `src/App.jsx`

**Done:**
- Renamed `testResultContext` → `resultContext` state variable

**Remaining:**
1. Build result data in `handleRegionVictory()`:
```js
const elapsed = performance.now() - state.current.combatStats.startTime;
const minutes = Math.floor(elapsed / 60000);
const seconds = Math.floor((elapsed % 60000) / 1000);
const timeStr = `${minutes}m ${seconds}s`;

const enemiesSlainTypes = Object.entries(state.current.combatStats.enemiesSlain.types)
  .map(([name, count]) => ({ name, count }));

setResultContext({
  type: 'battle_win',
  time: timeStr,
  title: 'VICTORY',
  stats: {
    wavesConquered: state.current.combatStats.conqueredWaves,
    totalWaves: state.current.wave - 1,
    damageDealt: state.current.combatStats.damageDealt,
    enemiesSlain: {
      total: state.current.combatStats.enemiesSlain.total,
      types: enemiesSlainTypes
    }
  },
  resources: [
    { name: 'Honor', change: `+${state.current.earnedHonor}`, color: 'text-[#d4af37]' }
  ],
  impacts: []
});
```
